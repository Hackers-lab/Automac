Overview

The DCRC Automac V1.01 is an application designed to simplify the process of filtering and saving data from disconnection files. Whether you're a data analyst, researcher, or business professional, this tool provides a user-friendly interface for selecting filter criteria and saving the filtered results in Excel format.

This help file serves as a comprehensive guide to assist you in understanding and effectively using the DCRC Automac application. It provides step-by-step instructions, tips, and answers to common questions you may have during your usage.

Instructions

The following instructions will walk you through the process of using the DCRC Automac V1.01 application effectively:

  1. Click on the "Browse" button to select the disconnection file you want to filter. Ensure that the file is in a supported format and located on your local machine.
  2. Select the agency for which you want to filter the data from the dropdown list. The application provides a list of available agencies for you to choose from. This step helps narrow down the data set to the specified agency.
  3. Enter the desired amount in the corresponding input field. This allows you to filter the data based on a specific amount. For example, if you want to filter records with amounts greater than $100, enter "100" in the input field.
  4. Choose a class from the dropdown list to further refine the filtering criteria. The class selection helps categorize the data based on specific criteria, such as customer type or service category.
  5. Once you've set the filter criteria, click on the "Apply Filters" button to process the selected filters. The application will analyze the disconnection file and display the filtered results in the right pane.
  6. The filtered results will be displayed in a table format, showing the relevant data entries that match the specified criteria. Each row represents a distinct record, and each column represents a data field.
  7. If you need to search for specific entries within the filtered results, enter a search query in the search bar located above the table. The application will highlight the matching records, making it easier for you to locate the desired information.
  8. To export the filtered results to an Excel file, click on the "Export to Excel" button. The application will generate an Excel file containing the filtered data, which you can save to your preferred location on your computer.
  9. For any additional assistance or questions, refer to the "FAQ" section in this help file or contact our support team.

FAQ

Q: Which file formats does the DCRC Automac V1.01 support?

The DCRC Automac V1.01 supports disconnection files in CSV (Comma-Separated Values) and XLSX (Microsoft Excel) formats. Ensure that your file is in one of these formats to successfully load it into the application.

Q: How do I update the output directory for saving the filtered results?

To update the output directory for saving the filtered results, follow these steps:

  1. Locate the configuration file for the DCRC Automac application. The configuration file is typically named "config.ini" and is located in the application's installation directory.
  2. Open the configuration file using a text editor.
  3. In the file, find the line corresponding to the output directory setting.
  4. Update the output directory path with the desired path. Make sure to provide the full path and ensure that the directory exists.
  5. Save the configuration file and restart the DCRC Automac application for the changes to take effect.

Q: Can I export the filtered results as a PDF?

Currently, the DCRC Automac V1.01 only supports exporting the filtered results in Excel format. PDF export functionality is not available in this version of the application. You can save the Excel file and convert it to PDF using other software if needed.

Q: Can I filter the data based on multiple criteria simultaneously?

Yes, you can apply multiple filter criteria at the same time. Simply select the desired options from the dropdown lists and enter the corresponding values in the input fields. The application will apply all the selected filters to narrow down the data set accordingly.

Q: How long does it take to process the filtering operation?

The processing time depends on the size of the disconnection file and the complexity of the selected filter criteria. For small to medium-sized files, the operation is usually completed within a few seconds. However, larger files with extensive filtering requirements may take longer to process. Please be patient while the application analyzes and filters the data.

Q: Can I cancel or abort the filtering operation once it's initiated?

Currently, the DCRC Automac V1.01 does not provide a built-in cancel or abort feature for the filtering operation. Once the filters are applied, it's recommended to wait for the operation to complete. However, if you encounter any issues or unexpected behavior, you can close the application and restart it.

Q: Are the filter settings saved between sessions?

No, the application does not save the filter settings between sessions. Each time you open the DCRC Automac V1.01, you'll need to reselect the agency, enter the desired amount, and choose the class for filtering. This design ensures that you have control over the filtering process and allows flexibility when working with different data sets.

Q: Is there a limit on the number of records that can be filtered?

The application can handle a large number of records, but there might be practical limitations based on your computer's resources. Filtering a very large dataset could require significant memory and processing power. If you experience any performance issues or if the application becomes unresponsive, consider filtering the data in smaller batches or optimizing your filter criteria to reduce the number of records processed at once.

Q: Can I undo or revert the applied filters?

Currently, the application does not support an undo or revert feature for the applied filters. To remove the applied filters and view the complete dataset, you can close the DCRC Automac V1.01 and reopen it without selecting any filter criteria.

Q: How can I update the application to a newer version?

To update the DCRC Automac application to a newer version, please follow these steps:

  2. Download the latest version of the DCRC Automac setup file.
  3. Run the downloaded setup file and follow the on-screen instructions to install the newer version.
  4. During the installation process, you may be prompted to uninstall the previous version. Follow the prompts accordingly.
  5. Once the installation is complete, you can launch the updated version of the DCRC Automac application.

These shortcuts can be used within the application's input fields and menus.

Q: Can I print the filtered results directly from the application?

Yes, you can print the filtered results directly from the DCRC Automac V1.01 application. Once you have applied the desired filters and the results are displayed, you can use the print functionality of your web browser to print the page. Press Ctrl+P (or Command+P on Mac) to open the print dialog, review the print settings, and click the "Print" button to generate a physical or digital copy of the filtered results.

Q: How do I clear the search input fields and reset the filter criteria?

To clear the search input fields and reset the filter criteria, click on the "Reset" button located near the filter options. This action will clear all the entered values and reset the dropdown lists to their default selections. The application will revert to the initial state, allowing you to apply new filter criteria or perform a fresh search.

Q: Can I filter the data based on multiple agencies simultaneously?

No, the DCRC Automac V1.01 supports filtering the data for a single agency at a time. You can select one agency from the dropdown list to filter the disconnection records specific to that agency. If you need to filter data for multiple agencies, you can repeat the filtering process for each agency separately.

Q: How can I sort the filtered results?

Currently, the DCRC Automac V1.01 does not provide built-in sorting functionality for the filtered results. The application displays the filtered records in the order they appear in the original disconnection file. However, you can export the filtered results to an Excel file and use Excel's sorting features to arrange the data based on specific columns.

Q: Is there a way to save the filter settings for future use?

The DCRC Automac V1.01 does not have a built-in feature to save filter settings for future use. However, you can manually record the filter criteria you used by taking screenshots or writing them down. This way, you can refer back to your preferred filter settings when you reopen the application.

Q: Can I customize the columns displayed in the filtered results?

Currently, the DCRC Automac V1.01 does not provide customization options for the columns displayed in the filtered results. The application shows a predefined set of columns based on the disconnection file's structure. If you need specific columns or additional information, consider exporting the filtered results to an Excel file and customizing the columns within Excel.
